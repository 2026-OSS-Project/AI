# Model Performance Comparison

## 1. 실험 개요

본 문서는 분리수거 이미지 분류를 위한 CNN 모델의 성능을 비교하기 위해 작성되었다.

동일한 CNN 아키텍처를 기반으로 데이터셋 구성, 데이터 증강, 하이퍼파라미터 설정을 단계적으로 변경하며 실험을 진행하였으며, 각 실험 조건에 따른 성능 변화를 비교·분석하였다.

---

| 구분             | 내용                         |
| -------------- | -------------------------- |
| 데이터 종류         | 이미지 분류 데이터셋                |
| 클래스            | plastic, paper, can, vinyl |
| Train 데이터      |                            |
| Validation 데이터 |                            |
| Test 데이터       |                            |
| 이미지 전처리        | Resize, Normalization      |
| 입력 크기          | 224 × 224                  |

## 6. 최종 선정 모델

| 항목    | 내용                                            |
| ----- | --------------------------------------------- |
| 최종 모델 | CNN_v5                                        |
| 선정 이유 | 가장 높은 Test Accuracy와 안정적인 Validation Loss를 기록 |
| 주요 장점 | 데이터 증강과 하이퍼파라미터 최적화를 통해 일반화 성능 향상             |
| 한계점   | 데이터 수가 제한적이며 클래스 불균형 가능성이 존재                  |

---

## 7. 실험 결과 요약

* 데이터셋 확장을 통해 전반적인 분류 성능이 향상되었다.
* 데이터 증강 적용 후 과적합이 감소하고 일반화 성능이 개선되었다.
* 하이퍼파라미터 튜닝을 통해 Validation Loss를 안정적으로 감소시킬 수 있었다.
* 최종적으로 CNN_v5를 최종 모델로 선정하였다.
