# 클래스 기준표

| 클래스 | 설명 | 예시 |
| --- | --- | --- |
| plastic | 플라스틱 재질의 용기 및 제품 | 페트병, 플라스틱 컵, 플라스틱 용기 |
| paper | 종이류 및 종이 포장재 | 종이컵, 박스, 신문지 |
| can | 금속 캔류 | 음료 캔, 통조림 캔 |
| vinyl | 비닐 및 필름류 | 비닐봉지, 포장 비닐 |

※ 실제 분리배출 기준과 일부 차이가 있을 수 있으며, 본 프로젝트는 이미지 분류 모델 학습을 위한 기준을 사용한다.

---

**특이사항**

plastic과 vinyl은 실제 환경에서 혼동이 자주 발생하기 때문에, 형태와 재질 특성을 기준으로 구분하였다. 본 기준은 모델 학습 과정에서 라벨 일관성을 유지하기 위해 사용되었으며, 공개 데이터셋 클래스와의 매핑에도 동일하게 적용하였다.
