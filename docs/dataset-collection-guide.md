# 데이터 수집 가이드

본 프로젝트에서는 폐기물 이미지 분류 모델의 학습 성능 향상과 데이터 일관성 확보를 위해 데이터 수집 기준을 정의하였다.

## 1. 수집 대상 클래스

- plastic
- paper
- can
- vinyl

## 2. 촬영 규칙

데이터 간 배경 편차를 최소화하고 객체 특징 학습에 집중할 수 있도록 모든 이미지는 흰색 A4 용지를 배경으로 사용하였다.

- 하나의 이미지에는 하나의 객체만 포함한다.
- 객체가 이미지의 약 60~80%를 차지하도록 촬영한다.
- 너무 멀리서 촬영하지 않는다.
- 흐린 사진은 사용하지 않는다.
- 손으로 객체를 가리지 않는다.
- 지나친 필터 및 보정을 적용하지 않는다.
- 배경은 흰색 A4 용지로 통일한다.

## 3. 촬영 각도

모델의 일반화 성능 향상을 위해 다양한 각도에서 데이터를 수집하였다.

- 정면
- 측면
- 위쪽
- 기울어진 각도

## 4. 객체 상태

실제 분리배출 환경에서는 객체가 손상되거나 변형된 경우가 많기 때문에 다음 상태를 포함하였다. 

- 깨끗한 상태
- 구겨진 상태

## 5. 파일 형식

허용 파일 형식:

- .jpg
- .jpeg

## 6. 파일명 규칙

파일명 형식:

[class]_[번호].jpg

예시:

- plastic_001.jpg
- paper_023.jpg
- can_015.jpg
- vinyl_042.jpg

## 7. 제외 대상 기준

다음 이미지는 데이터셋에서 제외한다.

- 너무 흐린 사진
- 객체가 잘린 사진
- 여러 객체가 겹친 사진
- 지나치게 어두운 사진
- 해상도가 낮은 사진
