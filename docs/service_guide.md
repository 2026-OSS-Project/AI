# Streamlit 서비스 가이드

## 서비스 개요

본 서비스는 사용자가 분리수거할 물건 이미지를 업로드하면, AI 예측 결과와 분리배출 방법을 화면에 출력하는 Streamlit 기반 웹 서비스이다.

현재 단계에서는 실제 AI 모델 연결 전이므로 임시 예측값을 사용하고 있으며, 추후 학습된 모델 파일과 inference 코드를 연결할 예정이다.

## 주요 기능

- 이미지 업로드
- 업로드 이미지 미리보기
- AI 예측 결과 출력
- 예측 신뢰도 출력
- 카테고리별 분리배출 방법 안내
- Colab 기반 실행 지원
